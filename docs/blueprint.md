# **App Name**: Eventide Platform

## Core Features:

- Admin Dashboard: Provides an overview of upcoming, ongoing, and past events with key metrics and visualizations.
- Event Creation Wizard: Facilitates event creation with a multi-step form, progress bar, and draft auto-saving.
- Personalized Event Recommendations: Uses a tool powered by generative AI to suggest events to users based on their past activity and preferences, to maximize the relevance of displayed events.
- Registration and Ticketing: Manages user registration and ticket generation with payment flow integration and digital QR ticket creation.
- Real-Time Engagement Tools: Includes live chat, Q&A, and polling features for real-time user interaction during events.
- Role-Based Access Control: Firebase custom claims gatekeep access to different parts of the app based on user roles (Super Admin, Admin/Organizer, User/Attendee).
- QR Code Check-In: Allows event organizers to scan QR codes for attendees for streamlined check-in

## Style Guidelines:

- Primary color: Deep sapphire (#3B52D4) for a sense of trust and futurism.
- Background color: Almost-white with a hint of sapphire (#F2F4FF) to support the 'liquid glass' aesthetic.
- Accent color: Electric purple (#BE6DCE) to highlight interactive elements and calls to action.
- Body font: 'PT Sans', sans-serif, for a modern look with a touch of warmth. Headline Font: 'Space Grotesk', sans-serif, for a computerized feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, glowing icons to complement the 'liquid glass' aesthetic.
- Implement a card-based layout with fluid animations for a modern and engaging user experience, using principles from Apple's 'Liquid Glass' design.
- Subtle, fluid animations on interactions to enhance user engagement and provide feedback.
